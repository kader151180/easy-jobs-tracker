# 🚀 Easy Jobs Tracker

A desktop application to manage your job search efficiently.

---

## 📌 Overview

Easy Jobs Tracker helps you organize job applications, track progress, and manage job search activities in one place.

Instead of using spreadsheets, this tool provides a clean UI, built-in browsing, and analytics dashboard.

---

## ✨ Features

* 📋 Track job applications (status, dates, notes)
* 🌐 Built-in browser for job searching
* 🔎 Save job searches from multiple platforms
* 📊 Dashboard with charts (status & provider insights)
* 📗 Export job data to Excel
* 🧠 Smart duplicate detection

---

## 🧰 Supported Job Platforms

* LinkedIn
* Indeed
* ZipRecruiter
* Glassdoor
* Monster
* FlexJobs
* CareerBuilder
* Snagajob
* USAJobs
* Dice

---

## 📦 Installation

1. Download the installer from Releases
2. Run:

```
EasyJobsTrackerSetup.exe
```

3. Follow installation steps
4. Launch from Desktop or Start Menu

---

## ⚠️ Notes

* Windows may show a security warning (Unknown Publisher)
  → Click **More Info → Run Anyway**

* Data is stored locally on your machine
